# Dizgruntled

## Setup

1. Install `node` from https://nodejs.org/en/download/
2. Install `yarn` using `npm install yarn`
3. Run `yarn` to install the dependencies

## Running

1. Run `yarn dev` in the `client` directory
2. Visit `https://localhost:3000/`
